    # 乘机流程

    1.办理行李托运，换取登机牌

托运行李：凭机票或本人有效身份证件到相应的值机柜台(即吉祥柜台)办理乘机和行李托运手续，领取登机牌。

无托运行李：在自助值机上打印登机牌选择座位即可(可选靠窗, 即A和F)。

2.进行安全检查

取完票之后, 需要找到安全检查的入口，提前准备好机票、登机牌和有效身份证件，并交给安全检查员查验。为了飞行安全，旅客必须从安全门通过安全检查，随身携带的行李物品必须经 X 光机检查。

3.机场候机，准备登机

检查完成之后, 按照机票上的候机站台是多少号, 然后根据路标指引走过去对应的候机台, 然后对一下航班出行信息, 然后就找个座位坐着。一般会提前15-30分钟左右登机。

4. 登机完毕

登机完成之后, 就找到对应的座位坐下, 然后系好安全带, 然后安静的看看风景, 飞机起飞和降落会有失重和超重的感觉, 待飞机平稳飞行的时候, 会有小食和一瓶水。

    # 乘机注意事项

    1. 因为安全规定, 不能携带液体上飞机, 所以买的水记得喝完, 安检完里面会有自助饮水机

2. 单瓶 < 100ml的洗漱护肤品(指的是瓶子的容量只能小于100ml, 100ml也不行), 然后水乳可以用我的, 面膜也可以, 口红、粉底那些, 只要不是液体，那些是可以带的。如果需要携带液体的话, 就得去买那种飞机携带化妆品的小瓶子(网上有卖)。

3. 充电宝是可以带的, 最多两个。

4. 牙膏>100ml 不能携带。

5. 20寸的行李箱是可以携带上去飞机, 不需要托运的。

6. 免费托运的最大重量为20kg
